workbox-routing.RegExpRoute

class   static

RegExpRoute is a helper class to make defining regular expression based Routes easy.

Please note that this only needs to match any part of the URL for same-origin requests. For requests against third-party servers, you must define a RegExp that matches the start of the URL.

For example,

// Matches all local styles
new RegExpRoute(new RegExp('/styles/'), ...);

while

 // Matches third-party styles
new RegexpRoute(new RegExp('https://example\.com/styles/'), ...);
